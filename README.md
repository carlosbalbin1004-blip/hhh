
# SKIINBOX — Sitio web (Next.js + Tailwind)

Este repositorio es un **scaffold** listo para subir a GitHub. Inspirado en la estética de Brunello Cucinelli: minimalismo, lujo sobrio y transiciones suaves.

## Qué incluye
- Next.js app (Single page) con componente principal `components/Skiinbox.jsx`.
- TailwindCSS configurado (archivos de configuración incluidos).
- Estructura y contenido (Home, Cómo funciona, Planes con precios, Lookbook, Test de estilo).
- Imágenes placeholder en `public/assets/` — reemplaza por tus URLs de Drive o sube imágenes aquí.
- Scripts: `npm run dev`, `npm run build`, `npm start`.

## Pasos para ejecutar localmente
1. `npm install`
2. `npm run dev`
3. Abrir `http://localhost:3000`

## Notas
- Reemplaza imágenes en `public/assets/` con tus links públicos (Drive o CDN).
- Configura Stripe o la pasarela local para pagos.
- Importa las tipografías Playfair Display y Montserrat en `pages/_document.js` o en tu `globals.css`.
